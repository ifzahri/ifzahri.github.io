November 6, 2020

#JUARAGCP - Program edukasi Google Cloud

JuaraGCP merupakan program yang dirancang untuk mengajak developers Indonesia menyelesaikan pelatihan tentang komputasi awan di Qwiklabs. Pelatihan ini juga dapat mempersiapkan peserta untuk melanjutkan perjalanan pembelajaran dalam meraih sertifikasi GCP. Disini, saya akan membagikan pengalaman saya dalam mengikuti JuaraGCP Season 3 & 4

Season 3

Season 3 saya berhasil lewati dalam 5 hari saja untuk mengerjakan 13 quest. Saya menemukan banyak sekali atau mungkin hampir semuanya sangat sulit untuk menyelesaikan quest-questnya karena berbenturan dengan jadwal maintenance Qwiklabs (biasanya lab yang bermasalah akan dimaintenance selama 1-2 minggu). Disini saya tidak menemukan kesulitan tertentu

Tema season 3 ini topiknya Machine Learning, DevOps & Infrastructure dan Application Development. Topik yang benar-benar menggugah wkwkwk. Anyway, disini saya benar-benar terbuka matanya melihat pemrosesan dan cara kerja sebuah aksi atau task di cloud yang benar-benar berbeda dibandingkan dengan proses di standalone.

Saya yang pertama kali pegang cloud cukup bingung awalnya cara mengoperasikannya. Namun berkat instruksi yang detail dari Qwiklabs, Perlahan-lahan saya membentuk muscle memory akibat seringnya start ulang lab wkwk. Topiknya sangat menarik dimana saya belajar deploy WebApp di GCP, membangun Serverless dengan GCP, membuat scaling infrastruktur dengan microservices dan menghubungkannya dengan CI/CD. Wah ruwet :))))

Kebetulan karena bulan puasa, maka saya jadikan event ini jadi teman menghabiskan waktu, dan hebatnya lapar haus pun tak terasa :v. Mungkin karena saya sangat mendalaminya hingga kadang tak terasa sudah adzan ashar bahkan maghri dengan start dari pagi. Dengan ketekunan saya demi ~swag~, dalam 5 heri beres wkwkwkw, dan saya pun berhak mendapat swag. Tim #JuaraGCP Pun gerak cepat kirim swag, tidak sampai satu bulan stop season sudah sampai dirumah swagnya~~

Season 4

Setelah senang mendapat tas, tumbler, dan topi di Season 3, jiwa-jiwa swaghunter saya bangkit kembali ketika Season 4 diumumkan, dan waw swagnya hebat season ini

Ya sudah langsung gas saja daftar dan ngerjain. Topiknya season kali ini tentang Data, AI, ML, Infrastructure and Application Development. Sangat super sekali. Namun tentu saja swag yang hebat diimbangi dengan tantangan yang hebat juga.

Data, AI and Machine Learning

  • Perform Foundational Data, ML, and AI Tasks in Google Cloud
  • Engineer Data in Google Cloud
  • Insights from Data with BigQuery
  • Integrate with Machine Learning APIs
  • Explore Machine Learning Models with Explainable AI
  • Automate Interactions with Contact Center AI
  • Build Interactive Apps with Google Assistant
  • BigQuery for Machine Learning
  • Building Advanced Codeless Pipelines on Cloud Data Fusion
  • Data Catalog Fundamentals

Application Development & Infrastructure

  • Build a Website on Google Cloud
  • Deploy and Manage Cloud Environments with Google Cloud
  • Set up and Configure a Cloud Environment in Google Cloud
  • Build and Secure Networks in Google Cloud
  • Cloud Architecture: Design, Implement, and Manage
  • Build and Manage APIs with Apigee
  • Implement DevOps in Google Cloud
